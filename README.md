# GitHub Helper

A collection of modern developer tools built with Next.js, TypeScript, Tailwind CSS v4, and shadcn/ui.

## Live Demo

https://tool-stack-kappa.vercel.app/

---

## Features

### GitHub Tools

- Commit Generator
- Diff Checker

### Developer Utilities

- Regex Tester
- JWT Debugger
- Base64 / URL Encoder Decoder

### Coming Soon

- JSON Formatter
- Markdown Preview
- SQL Formatter
- Hash Generator
- UUID Generator
- URL Parser
- CSV Viewer
- Image Converter

---

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Lucide Icons
