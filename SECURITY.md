# Security Policy

Thank you for helping keep **Tool Stack** secure.

## Supported Versions

At this time, only the latest version of the project is actively supported with security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅ Yes     |
| Older versions | ❌ No      |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public GitHub issue**.

Instead, report it privately by emailing:

**📧 [derupanjan2021@gmail.com](mailto:derupanjan2021@gmail.com)**

Please include the following information:

* A clear description of the vulnerability.
* Steps to reproduce the issue.
* The potential impact.
* Any suggested fix (optional).

You can expect:

* An acknowledgment within **24 hours**.
* Regular updates during the investigation.
* Credit for the discovery (if desired) after the issue is resolved.

## Responsible Disclosure

Please allow time for the vulnerability to be investigated and fixed before publicly disclosing it.

We appreciate your efforts to help keep the project and its users safe.

Thank you for contributing to the security of Tool Stack.
